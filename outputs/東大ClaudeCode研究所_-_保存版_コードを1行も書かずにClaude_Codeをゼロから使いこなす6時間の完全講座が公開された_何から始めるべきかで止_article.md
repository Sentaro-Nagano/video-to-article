## Claude Codeの概要
Claude Codeは、AIアシスタントを構築するためのツールです。Cloud Codeは、Claude Codeのエンジンです。Firecrawlは、ウェブサイトをLLM用のデータに変換できるツールです。MCPサーバーは、Firecrawlの機能をCloud Codeに統合するために使用されます。

## Claude Codeの設定手順
Claude Codeの設定手順は以下の通りです。

1.  フォルダ構造の作成とGitリポジトリの初期化
2.  インタビュー
3.  コミュニケーション設定
4.  ツールの設定
5.  スキルの作成

## Cloud Codeの機能
Cloud Codeの機能は以下の通りです。

*   フォルダ構造の作成と管理
*   Gitリポジトリの管理
*   インタビューの回答を基にフォルダとファイルの作成
*   コミュニケーションスタイルの設定
*   スキルの作成と実行
*   .envファイルの作成とAPIキーを入力する
*   プロジェクト管理ツールに接続する
*   研究スキルの作成と実行

## Firecrawlの機能
Firecrawlの機能は以下の通りです。

*   ウェブサイトのコンテンツをスクラップして取得することができます。
*   ウェブサイトの構造をマップすることができます。
*   ウェブサイトをクロールすることができます。
*   ウェブサイトを検索してデータを取得することができます。

## MCPサーバーの使用
MCPサーバーは、Firecrawlの機能をCloud Codeに統合するために使用されます。MCPサーバーは、Nateの自然言語のリクエストに基づいて、Firecrawlの機能を選択して実行します。

## Cloud CodeとBlotatoの使用
Cloud Codeを使用して、YouTube動画をLinkedIn、Instagram、Xに投稿するためのAIソーシャルメディアマネージャーを作成することができる。Blotatoを使用して、YouTube動画のトランスクリプトを抽出し、AIが投稿用のコンテンツを生成することができる。

## GWS CLIの概要
GWS CLIはGoogle Workspace CLIのことであり、Google Workspaceの機能を操作するためのコマンドラインインターフェースです。GWS CLIはオープンソースのGoogle Workspace製品であり、完全に無料です。

## GWS CLIの利点
GWS CLIの利点は以下の通りです。

*   一つのインターフェースでGoogle Workspaceの機能を操作できる。
*   JSON形式のレスポンスを提供するため、AIエージェントが扱いやすい。
*   自動ディスカバリー機能により、CLIは常に最新の状態を維持する。
*   プリビルドのスキルを使用して、複数のステップのワークフローを実行できる。

## 必要なスキルとツール
必要なスキルとツールは以下の通りです。

*   フロントエンドデザインスキル
*   ビデオをウェブサイトに変換するスキル
*   Cloud Code
*   Visual Studio Code
*   Anthropic
*   Nano Banana
*   Key.ai
*   Cling 3.0
*   FFmpeg

## 手順
手順は以下の通りです。

1.  Cloud Code をインストールし、Visual Studio Code に追加する。
2.  Anthropic の有料プランにサインインする。
3.  3D ウェブサイトを作成するために必要なスキルとツールを準備する。
4.  Nano Banana を使用して、プロフェッショナルなスタジオ画像を作成する。
5.  Cling 3.0 を使用して、画像を動画に変換する。
6.  Cloud Code にビデオをアップロードし、ウェブサイトを作成する。
7.  Cloud Code にウェブサイトの詳細を入力し、実装を開始する。
8.  FFmpeg を使用して、動画からフレームを抽出する。
9.  Cloud Code がウェブサイトを作成し、ローカルでテストする。
10. ウェブサイトを公開する。

## GitHubとVercelを使用してウェブサイトをデプロイする
GitHubとVercelを使用してウェブサイトをデプロイする手順は以下の通りです。

1.  GitHubにアカウントを作成する。
2.  Vercelにアカウントを作成する。
3.  CloudCodeを使用してGitHubとVercelを連携する。
4.  GitHubにコードをプッシュする。
5.  Vercelにデプロイする。

## 5つのハック
5つのハックは以下の通りです。

*   1. フロントエンドデザインスキルを使用して、ロゴ、色、フォントを含む完全なランディングページを作成する
*   2. スクリーンショットループを使用して、AIが自動的にスクリーンショットを撮り、参照サイトと比較して改善する
*   3. 他のサイトを参考にしたクローンを使用して、AIが自動的にクローンサイトを作成する
*   4. Bypass Permissions Modeを使用して、AIが自動的にコマンドを実行する
*   5. 個別のコンポーネントを使用して、ウェブサイトをより個性化できる
